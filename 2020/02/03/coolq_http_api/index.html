HTTP API 文档

NoneBot挂不稳 转而使用Fask版的SDK
其实什么版本都差不多
酷Q登录后 收到消息 就会通过HTTP POST上报
上报地址就是插件配置文件的 post_url
有个服务器接收这个事件 处理过后返回就ok

环境

  • CentOS 7
  • Python 3.7.5
  • NoneBot 1.3.1
  • Python 必须 >= 3.6 其他好像无所谓
